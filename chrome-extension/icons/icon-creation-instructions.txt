ICON CREATION INSTRUCTIONS
==========================

You need to create 3 icon files with the placeholder logo:
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)  
- icon-128.png (128x128 pixels)

Use the placeholder logo image provided and resize to these dimensions.

For now, you can use any tool like:
- Preview/Photos (Mac/Windows)
- GIMP/Photoshop
- Online tools like cloudconvert.com

Or use command line with ImageMagick:
  convert logo.png -resize 16x16 icon-16.png
  convert logo.png -resize 48x48 icon-48.png
  convert logo.png -resize 128x128 icon-128.png

Make sure icons have:
- Transparent background (PNG)
- Square dimensions
- Clean scaling (use bicubic/lanczos for best quality)
